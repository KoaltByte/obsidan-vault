- Es la primera fase de cualquier prueba de penetración.
- Consiste en recopilar información de una persona,. una web o sistema.

# Recopilación pasiva
- La [[Recopilación pasiva]] consiste en recopilar la mayor cantidad de información posible sin vernos comprometidos directamente con el objetivo.
- Se hace uso de recursos públicamente disponibles o accesibles para la obtención de información.
- También se puede hacer una análisis desde la perspectiva de un usuario final, a través del navegador. 
- Ejemplos: Sitio web de la empresa, google dorks, uso de OSINT, registros DNS, etc. 
## Que se busca en la recopilación pasiva?
1. Identificar direcciones IP e información sobre DNS.
2. Identificar nombres de dominio o información de empleados.
3. Identificar direcciones de e-mail o perfiles de redes sociales.
4. Identificar tecnologías utilizadas en el sitio objetivo.
5. Identificar subdominios.


# Recopilación activa
- Recopilación de información mediante una participación activa con el objetivo
- Ya que se interactúa directamente con el sistema de destino, lo que significa una comunicación directa, para este tipo de recopilación se requiere autorización previa por parte de la empresa.
- Ejemplos: Escaneo de puertos a la dirección IP del servidor.
## Que se busca en la recopilación activa?
1. Descubrir puertos abiertos de los sistemas objetivo.
2. Aprender sobre la infraestructura interna del objetivo como la red y la organización.
3. Enumeración de información de los sistemas.
