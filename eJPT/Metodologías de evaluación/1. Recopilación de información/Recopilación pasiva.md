## 1. Obtención de la IP del servidor
Para obtener la dirección IP del servidor que resuleve al dominio de cualquier web se hace uso del comando ``host <nombreDominio.com>

![[Pasted image 20251008132515.png]]

Al ejecutar el comando a nuestra página web objetivo, notamos que nos da dos direcciones IP, lo cual significa que esa web tiene de por medio un [[proxy]]
