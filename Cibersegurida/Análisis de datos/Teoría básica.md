# Pasos para un análisis eficaz
1. Realizar la pregunta correcta: Qué es lo que el negocio necesita? Por que estamos observando esto?, qué decisiones se tomaran al finalizar el análisis?
2. Obtención de datos: Recolectar toda la información que pueda resultar útil para el análisis.
3. Limpieza y preparación de los datos (Data Cleaning / Wrangling): Esto es necesario para eliminar el ruido de los datos -> Valores faltantes, duplicados, sin formato o con formato incorrecto, etc.
4. Análisis y exploración:  Busqueda de patrones, tendencias, ==insights== (hallazgos clave). 
5. Visualización y comunicación de los resultados: Utilizando herramienta como power BI o Tableau.
