# WAN 
---
- La WAN (red de área extensa) se conoce comúnmente como `The Internet`. Al trabajar con equipos de red, solemos tener una dirección WAN y una dirección LAN
	- WAN es la dirección a la que generalmente se accede a través de Internet.
- Una WAN es simplemente un <mark style="background: #D2B3FFA6;"> gran número de redes LAN conectadas</mark>. Muchas grandes empresas o agencias gubernamentales tienen una "<mark style="background: #D2B3FFA6;">WAN interna</mark>" (también llamada intranet, red Airgap, etc.).

# LAN / WLAN
---
- Las redes LAN (redes de área local) y WLAN (redes de área local inalámbricas) suelen asignar direcciones IP designadas para uso local.
- No hay ninguna diferencia entre una LAN y una WLAN, salvo que las WLAN permiten transmitir datos sin cables.

# VPN
---
Existen tres tipos de [[VPN]] o <mark style="background: #D2B3FFA6;">Virtual Private Networks</mark> pero los tres tienen el mismo objetivo: hacer que el usuario se sienta como si estuviera conectado a una red diferente.

## VPN site-to-site
Tanto el cliente como el servidor son dispositivos de red, generalmente [[Routers]] o [[Firewalls]], y comparten rangos de red completos. Esto se usa comúnmente para conectar redes empresariales a través de Internet, lo que permite que varias ubicaciones se comuniquen a través de Internet como si fueran locales.

## VPN de acceso remoto 
Estas VPN´s  implican que la computadora del cliente crea una interfaz virtual que se comporta como si estuviera en la red de un cliente.

## VPN SSL
Se trata básicamente de una VPN que se ejecuta dentro de nuestro navegador web y es cada vez más común a medida que los navegadores web se vuelven más capaces de hacer cualquier cosa. Normalmente, transmiten aplicaciones o sesiones completas de escritorio a tu navegador web.
