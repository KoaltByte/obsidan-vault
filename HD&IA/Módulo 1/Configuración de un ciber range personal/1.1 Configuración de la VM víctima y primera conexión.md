# Comprobando conexión entre máquina atacante y máquina víctima (ping)

- Cuando dos o más dispositivos se encuentran conectados dentro del mismo segmento de red, se pueden comunicar entre sí, una forma rápida para verificar que haya comunicación es utilizando el comando `ping`
- El comando `ping` utiliza el [[Protocolo ICMP]]  

````bash
ping -c 4 192.168.186.129
````
- `ping`: Comando que usa ICMP
- `-c 4`: Parámetro que limita el envío de paquetes a 4.

- La máquina atacante pregunta a la máquina víctima "Estás ahí?"
	*VM Kali* ---> **send pack  ICMP Echo Request** ---> *VM metasploitable 2* 
- La máquina víctima responde de regreso "Si, estoy aquí"
	*VM Kali* <--- **send pack  ICMP Echo Reply** <--- *VM metasploitable 2* 
## Relevancia de ciberseguridad
- Los atacantes y hacker eticos utilizan `ping` como el primer paso para el [[Reconocimiento de información]], de esta manera saben que máquinas se encuentran activas en la red.
- Cuando una máquina no responde puede estar apagada o tener un firewall que esta bloqueando ICMP.
