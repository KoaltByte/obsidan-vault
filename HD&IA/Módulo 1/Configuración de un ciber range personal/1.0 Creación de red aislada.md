> - Se ha descargado y creado la VM de Kali en el virtualizador.

# Conexión host-only en la VM
- Dependiendo el virtualizador, se debe de configurar la VM con un adaptador de red en modo host-only.
	- Para VirtualBox, se debe de crear de forma manual la red host-only (adaptador solo anfitrión).
	- Para VMware, la tiene creada por defecto, solo se debe de seleccionar en las configuraciones de red de la VM.

## Conceptos clave (Redes)
- ==Adaptador de red virtual== : Tarjeta de red simulada por el virtualizador. 
- ==Red Host-Only== : Este tipo de red es como si se tuviera un switch de red virtual (no físico), donde se conectan todas aquellas máquinas virtuales  que configuremos con este red. De esta manera se crea un LAN privada. 
	- La creación de una LAN privada permite tener un entorno de pruebas seguro y aislar nuestro equipo anfitrión y nuestra red LAN de todo el laboratorio.
