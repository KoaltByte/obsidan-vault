# Elementos de la comunicación
Los elementos principales para que la comunicación entre computadoras a través de una red pueda existir, son:
1. Emisor
2. Receptor
3. Datos
4. Medios

# Comunicación
Se basa en la transferencia de mensajes, se hacen uso de protocolos con paradigmas orientados a capas (OSI, TCP/IP), ya que se debe asegurar el transporte confiable de los datos.

Otra forma de comunicación es la llamada a <mark style="background: #FFB86CA6;">procedimientos remotos (RPC) </mark> , en esta un proceso cliente que es ejecutado en una computadora, llama a un proceso ejecutado en otra.

## Categorías
### Capa física
La capa física contiene todos los elementos que requiere una computadora para comunicarse con otras computadoras dentro de la red, sirve para la transmisión de los datos por medios físicos, por ejemplo, tarjetas de red, cables, etc.
### Capa lógica
La capa lógica consta de normas que son usadas para construir los llamados protocolos de
comunicación, los cuales se asemejan al uso de los idiomas en las personas. Si dos personas no hablan el mismo idioma o si no respetan sus turnos para hablar, les sería imposible comunicarse entre sí.
