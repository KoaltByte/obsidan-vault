# Definición
Esta capa de _software_ funciona como una capa de traducción oculta para permitir la comunicación, intercambio de recursos y la administración de datos en aplicaciones distribuidas.
![[Pasted image 20251020001653.png]]

# Características
Los servicios del sistema operativo y protocolos de comunicaciones se independizan; asimismo, permite la interacción de distintos servicios en un sistema.

# Tipos de Middleware
---
## Middleware general
Son aquellos que requieren todos los clientes y servidores, como es el caso de un _software_ de comunicación que usa TCP/IP, u otro que forma parte del sistema operativo o que almacena archivos distribuidos de autenticación.
## Middleware de servicios
Es todo aquel _software_ que se asocia únicamente a un solo servicio, como es el caso de uno que se usa para la conexión de una BD a una red cliente/servidor (ODBC), o la tecnología CORBA, que permite que los objetos distribuidos en distintos lenguajes puedan interactuar en una misma red para el intercambio de mensajes, u otro _software_ que se emplea para tener conexiones seguras, por ejemplo..
